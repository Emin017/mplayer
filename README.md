## Features

- 🎵 Audio playback
- 🎨 Modern SwiftUI interface
- 🖥️ macOS support

## Requirements

- Xcode 15.0+
- macOS 14.0+
- Swift 5.9+

## Build and Run

1. Clone the repository:
```bash
git clone https://github.com/Emin017/mplayer.git
cd mplayer
```

2. Open the project in Xcode:
```bash
open mplayer.xcodeproj
```

3. Select a target device and run the project
