## 开发环境要求

- Xcode 15.0+
- macOS 14.0+
- Swift 5.9+

## 构建和运行

1. 克隆项目：
```bash
git clone https://github.com/Emin017/mplayer.git
cd mplayer
```

2. 使用 Xcode 打开项目：
```bash
open mplayer.xcodeproj
```

3. 选择目标设备并运行项目
